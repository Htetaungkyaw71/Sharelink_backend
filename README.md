# ShareLinks Backend

Link-sharing app

## Description

Link-Sharing app. Users can create, read, update, delete links, and see previews in the mobile mockup. Authentication included.

## Getting Started
```
git clone git@github.com:Htetaungkyaw71/Sharelink_backend.git
```
